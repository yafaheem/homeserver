Curl / direct upload examples

# simple (no auth)
curl -F "file=@/path/to/photo.jpg" http://YOUR_PC:5000/upload

# token as header
curl -H "X-UPLOAD-TOKEN: yourtoken" -F "file=@photo.jpg" http://YOUR_PC:5000/upload

# token as query
curl -F "file=@photo.jpg" "http://YOUR_PC:5000/upload?token=yourtoken"
